Mac软件配置

Posted on 2019-02-01 | In Computer settings
关于Mac系统上的一些软件以及相关配置。

前言

曾经我也是一个爱折腾的娃，随着年纪增大，慢慢的不爱折腾了，开始记录一些折腾的事。算是为了自己以后配置MAC而准备的。

Syncthing

至从resilio sync被墙以后，现在无精力一一配置服务器中转，因此转到开源软件Syncthing。
Syncthing是一个好东西，可以让NAS，MAC、Windows和Linux全平台进行同步。

Keepass

对于密码管理我用的是KeePass软件，对于在线在可以通过KeeWeb来配置，目前可以在线版Online Web App和Onedrive来配合使用.

IINA

目前比较喜欢的视频播放软件IINA。

Zotero

所以科研工作者都有自己的文献管理软件，我从Jabref转到Zotero。
